Introduction

Welcome to the documentation for the Layer Viewer page. This document will guide you through the various features and functionalities available on this page.

Getting Started

To begin using the Layers page, follow these steps:

  1. Select a Container: Choose which container you want to view the layers for.
  2. Understanding the Information: Layer x-y means that the ordering of the x-th through y-th layers is interchangable. The Packages field lists the packages explicitiy included in the layer. The New Data field of layer N gives the file size in megabytes that is included with layer N, but is not with layers 0 through N-1. The Package File Size field gives the combined file size of the packages explicitiy included in the layer. The Total Recursive File Size field gives the combined file size of the packages explicitiy included in the layer, and the packages that they recursively refernece. The Recursively Accounted For Packages field of Layer N lists the packages included with layer N, but not with layers 0 through N-1.
